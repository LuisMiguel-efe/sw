# **App Name**: Electron Cloud Portfolio

## Core Features:

- Project Showcase: Display a comprehensive list of academic projects, each with a title, description, tech stack, and link to the repository or live demo.
- Skills Visualization: Visually represent technical skills (automation, cloud computing, etc.) using charts or graphs to showcase proficiency levels.
- About Me Section: Detailed section highlighting the engineer's background, interests (automation, cloud computing), and career goals.  Incorporate a professional headshot.
- Contact Form: An interface for visitors to send messages directly to the engineer.  Collects name, email, and message.
- Resume/CV Download: Offer a downloadable version of the engineer's resume or CV in PDF format.
- AI-Powered Project Suggestion Tool: Based on user input (e.g., "show me projects using AWS Lambda"), use an AI tool to display matching projects or recommend relevant skills to explore.

## Style Guidelines:

- Primary color: Deep blue (#1E3A8A), reflecting reliability and technical expertise.
- Background color: Light gray (#F0F0F0), providing a clean and modern backdrop.
- Accent color: Teal (#26A69A), suggesting innovation and automation.
- Headline font: 'Space Grotesk' (sans-serif), for a modern and technical feel; Body font: 'Inter' (sans-serif) to improve readability on longer texts.
- Use a consistent set of minimalist icons to represent different technologies and skills.
- Implement a clean, grid-based layout that is easy to navigate and showcases projects effectively.
- Subtle animations (e.g., project loading, skill chart transitions) to enhance user experience.
